Alert, Prompt dan Confirm

● JavaScript memiliki fitur yang bernama
1. alert
2. prompt
3. confirm

● Alert digunakan untuk memberi peringatan berupa popup text di browser

● Prompt digunakan untuk meminta input string dari pengguna browser dalam bentuk popup input text

● Sedangkan confirm digunakan untuk meminta input boolean dari pengguna browser dalam bentuk
popup input pilihan:
ok : true
cancel : false
